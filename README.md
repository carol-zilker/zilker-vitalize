# Zilker Vitalize

Site institucional da Zilker Vitalize — Massoterapia e Terapias Integrativas em São Miguel Paulista, São Paulo.

Réplica construída com [Next.js](https://nextjs.org) 16 (App Router), TypeScript e Tailwind CSS 4, a partir do site original publicado em v0.dev.

## Desenvolvimento local

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Build de produção

```bash
npm run build
npm start
```

Veja [DOCUMENTACAO.md](./DOCUMENTACAO.md) para detalhes sobre o que foi feito, estrutura do projeto e como fazer deploy.
